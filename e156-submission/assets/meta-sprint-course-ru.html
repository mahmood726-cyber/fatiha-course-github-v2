Quick Reference

DoD (Definition of Done)
Binary gate checklist. All items must pass before proceeding to next phase.
PICOS
Популяция, Вмешательство, Сравнительный анализ, Результаты, Дизайн исследования – основа для исследовательских вопросов.
Provenance
Журнал аудита, показывающий, откуда именно взялось число (например, «Таблица 2, стр. 1234»).
Red-Team
Daily adversarial checking by 2 rotating members to catch errors in extraction.
CondGO
Emergency protocol: 72 hours to fix a failed DoD gate or terminate with documented reasons.
Surrogate Outcome
A measurable marker (e.g., cholesterol) used as proxy for patient-important outcomes (e.g., death).
FREEZE (Day 34)
Блокировка области — после этого момента нельзя добавлять новые исследования.
I² (I-squared)
Heterogeneity statistic. Low <30%, Moderate 30-60%, High >60%.
Analysis Capsule
Полный воспроизводимый пакет: код, данные, инструкции. Любой может бегать и получать одинаковые результаты.
Consult Pack
Структурированный документ с вариантами, доказательствами и компромиссами для принятия командных решений.
PROSPERO
Международный реестр протоколов систематического обзора. Временные метки ваших предварительно указанных методов.
PRISMA
Руководство по составлению систематических обзоров. Включает блок-схему, показывающую выбор исследования.
